# Phase 4: Query And Identity Hardening

## Scope

Improve mock fidelity for queries and client identity.

## Requirements

- Add composite key helpers.
- Add range query support.
- Add partial composite key query support.
- Add conservative rich query support.
- Fail loudly for unsupported selector behavior.
- Make common Fabric `cid` helper flows work for MSP ID, ID, and attributes.

## Deliverables

- Composite key and query implementations.
- Identity provider compatible with common Fabric client identity usage.
- Tests for supported query operators and unsupported selector failures.
